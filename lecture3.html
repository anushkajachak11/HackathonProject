Lecture 3: Nested Lists and Tables:

Introduction:

This lecture focuses on two important HTML elements: lists and organizing complex data clearly on a webpage.

Types of Lists:

  • Ordered List(<ol>)-Displays items with numbers(1,2,3...).
  • Unordered List(<ul>)-Displays items with bullets(.).

Nested Lists:

A list inside a another list is called a Nested list. This helps to create multi-level categories.

   <ul>
      <li>Frontend:
            <ol>
               <HTML>
               <CSS>
               <JavaScript>
               <React></li>
            </ol>

      <li>Backend:
            <ul>
               <Django>
               <Python>
               <Java>
               <Expressjs> </li> 
            </ul>
   </ul>

Tables in HTML:

Tables are use to organize data into rows and columns.

  • <table>- Defines a table.
  • <tr>-Table Row.
  • <tr>-Table Data.
  • <th>-Table Header.

Advanced Table Features:

  • colaspan - Merge columns.
  • rowspan - Merge rows.
  • Sections - <thead>,<tbody>,<tfoot>.

Practical Uses:

Lists are used for navigation menus,steps and categories. Tables
are used for schedules,product features,comparison charts,and reports.

Summary:

  1. Lists organize related items(ordered/unordered)
  2. Nested Lists allow multi-level grouping.
  3. Tables organize data into rows and columns.
  4. Advanced tables use colspan,rowspan,thead,tbody,tfoot.
  5. Both lists and tables make webpages more redable and structured
