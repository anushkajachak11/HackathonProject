Lecture 5: HTML Forms:

The <form>Tag:

  • Main Container for all form elements.
  • Used to send data from user-server.
  • Attributes:
    • action: - URL where data is sent.
    • method: - GET (data visible in URL) or POST (data hidden in body)
    <form action ="/submit"
    method="post">
    </form>

The <input> Element:

  • Creates fields for user input.
  • Types:text,password,number,email,date,color,range,file,etc.
  • Attributes:
    • name - key sent to server.
    • placeholder - hint text inside box.
    • required - makes field compulsory.
    <input type="text" name="username" 
    placeholder="EnterName" required>

<label> Tag:

  • Provides descripsion to input field.
  • Connects to input using for(label) and id(input).
  • Improves usability and accessibility.
    <label for="fname">First Name:</label>
    <input type="text" id="fname" name="fname">

Submit Buttons

  • Used to send form data.
  • Two ways:
    • <input type="submit"> - simple
    • <button type="submit"> ... </button> - flexible, allows text/images
<input type="submit" value="Submit">
<button type="submit"><b>Submit</b> Form</button>

Radio Buttons:

  • Allow only ONE option from a group.
  • All radios in group share same name.
  • Each radio must have unique id and value.
<label>T-Shirt Size:</label><br>
<input type="radio" id="s" name="size" value="small">
<label for="s">Small</label><br>

<input type="radio" id="m" name="size" value="medium">
<label for="m">Medium</label>

Checkboxes:

  • Allow MULTIPLE selections.
  • Same name groups them together.
  • Each has unique id + value.

Textarea:

  • For long multi-line text input.
  • Attributes: rows (height),cols(width).
<textarea id="msg" name="message" rows="4" cols="40"></textarea>

Dropdown:(<select>)

  • Compact list of many options.
  • Uses <option> for each choice.
  • value is sent to server, visible text is shown to user.
<select id="country" name="country">
  <option value="">--Choose--</option>
  <option value="in">India</option>
  <option value="us">USA</option>
</select>

Useful Input Types:

  • password - hides typed text
  • number (with min, max) - restrict numeric range
  • date - calendar picker
  • color - color picker
  • range - slider control
  • file - upload files

Important Attributes:

  • id - unique identifier on page
  • name - data key for server
  • placeholder - hint text
  • required - makes input compulsory
  • value - default value
  • min/max - restrict numbers/dates
