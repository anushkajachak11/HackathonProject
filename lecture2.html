Lecture 2: Intoduction to HTML:

What is HTML?

HTML stands for HyperText Markup Language It is the backbone of every website on the internet.
HTML defines the structure and content of a webpage using tags.

Basic HTML Structure:

        <!DOCTYPE html>
        <html>
        <head>
             <title>My First Page</title>
        </head>
        <body>
            <h1>Hello Coder Army!!</h1>
            <p>This is My First Hacathon Project.</p>
        </body>
        </html>

Headings and Paragraphs:

  • Headings: <h1> to <h6> (largest to smallest).
  • Paragraph: <p> defines a block of text

Organizing Content:

Lists are used to group related items:

  • Ordered List(ol) -> Numbered list
  • Unordered List(ul) -> Bulleted list
  • List Item (li) -> Each point inside a list
        <ol>
           <li> Step 1 </li>
           <li> Step 2 </li>
        </ol>
        
        <ul>
           <li> Step 1 </li>
           <li> Step 2 </li>
        </ul>

Adding Links:

Anchor tag is used to link to other pages:

        <a 
        href="https://www.google.com">Visit Google</a>

Adding Visuals (Images):

Use the img tag:

        <img src="image.jpg" alt="My Image" Width="300">

Formatting the Page:

  • <br> -> Line Break
  • <hr> -> Horizontal Rule

Summary:

  1. HTML defines structure of webpages.
  2. Basic Tags: <html>,<head>,<body> <h1>- <h6> , <p>.
  3. Lists: <ol>, <ul>, <li>.
  4. Links: <a>.
  5. Images: <img>.
  6. Formatting: <br>, <hr>.

Common HTML Tags:

Tag Use
<h1> to <h6> Headings
<p> Paragraph
<a> Link
<img> Image
