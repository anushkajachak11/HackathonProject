HTML Media:

The web is more than just text and images. Modern webpages must be able to play videos and audio without
relying on old plugins like Flash or QuickTime. HTML5 solved this with the <video> and <audio> tags.

The <video> Tag:

        <video src="movie.mp4"></video>

This shows only the first frame of the video, without controls.

With Controls:

        <video controls height="300px">
        <Source src="movie.mp4" </video>

Now the user can play, pause, change volume, and fullscreen.

Common Attributes:

  • controls - adds default player controls.
  • width and height resize the player.
  • autoplay - starts automatically.
  • muted - starts with no sound.
  • loop - repeats after finishing.
  • poster - image shown before play.
    <video controls>
    <source src="video.mp4" width="500px" height="300px" autoplay muted loop poster="nature.jpg";</video>

The <audio> Tag:

The <audio> tag is used to play sound files like mp3 or wav.

    <audio controls>
    <source src="song.mp3" </audio>

It supports attributes like controls, autoplay, loop, and muted.

Multi-Page Website:

A professional website is never just one page. Instead, it contains multiple connected pages. These pages are linked with the help of navigation links.

  • Home Page
  • About us
  • Contact us
  • Lectures

These pages are connected through navigation links.

Example Of Navigation bar:

<nav>
<a href="index.html">Home Page </a>
<a href="about.html">About us</a>
<a href="contact.html">Contact us </a>
<a href="lecture.html">Lectures </a>
</nav>

Steps to Create Multi-Page Website:

  1. Create individual HTML pages (index, about, contact, lecture).
  2. Add a navigation bar on each page.
  3. Connect pages using the <a> tag.
  4. Open in a browser and test all links.
  5. Deploy the website to make it live.

Best Practices:

  • Always name your homepage as index.html.
  • Use lowercase filenames (about.html, contact.html).
  • Keep the same navigation bar on all pages.
  • Organize images and files in proper folders.

Deploying Your First Website:

Using Netlify

  1. Sign up at Netlify.
  2. Drag and drop your project folder.
  3. Instantly get a free live link.

Pro Tips:

  • Check all links before deployment.
  • Use meaningful page names and titles.
  • Keep file names short and simple.
  • Add alt text to images.
