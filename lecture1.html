Lecture 1: Intoduction to Web Development and Internet Working:

This lecture covers the fundamentals of the internet,IP Addressing ,MAC Address,Ports,DNS,and Data Transmission

What is internet?

The Internet is a global network of connected computers that share information.It is also called a "network of networks"

Computers communicate by sending binary data (0s and 1s) as signals

Diagram: How Internet Works

Why do we need Switches and Routers?

  • Switch:Connects devices inside a local network(office/home).
  • Router:Connects different networks together to form the Internet.

What is an IP Address?

An IP Address is a unique number assigned to every device on the Internet.

  • Public IP:Unique,used on the Internet(given by ISP).
  • Private IP:Used only inside local networks(eg.,192.168.x.x).

IPv4:32-bit address ,~4.3 billion devices. IPv6:128-bit address,almost unlimited devices.

What is MAC Address?

A MAC(Media Access Control) Address is a Permanent hardware ID of a network device.

Example: 3C:22:FB:A3:B4:C5

How to find?On Windows:ipconfig /all,On macOS Linux: ifconfig

What are the Port Numbers?

Port numbers identify which service or application is being used on a computer.

  • Well-Known(0-1023): HTTP 80,HTTPS 443,DNS 53
  • Dynamic (49152-65535):Temporary , Client-side use

What is DNS(Domain Name System)?

DNS is like the phonebook of the Internet.It translates domain names into IP addresses.

Steps of DNS Resoluation:

  1. Browser checks local cache.
  2. Asks DNS Resolver (ISP/Google DNS).
  3. Resolver queries Root -> TLD -> Authoritative Server
  4. Final IP address is returned (eg.,google.com -> 142.250.72.206)

How is data sent on the Internet?

Data is divided into small packets before transmission.

  • Packets travel separately and may take different routes.
  • TCP ensures packets arrive in correct order and without errors.

TCP/IP Protocol: Provides reliable communication for all users.
